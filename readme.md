# Learning Method

I used Claude to generate me a curriculum based on the College Courses I already took, what I already know and what I want to learn. In this case, I wanted to improve my ML Mastery.

I have found this method to be exetremely efficetive. Not just Claude, but any other good LLM, can work as a personal tutor and create lessons that personlized to you.

Here is the prompt I use if anyone else wants to stay using this method to study anything:
------------------------------------------------------------------------------------------------------------------------------
Create a curriculum that would allow me to learn about < Insert your Topic here, the more specific the better >. 
The curriculum should be weekly and provide resources each week to learn the week's topics. 

BACKGROUND INFO
Here are relevant courses and topics that I have already studies:
* Topic/Course 1
* Topic/Course  2
* and so on... (Make sure to enter everything revelant ot the topic. If you don't, the LLM assumes that you have no understanding and creates a curriculum that covers thing you have already studied) 

IMPORTANT:
* For Each Week, only give youtube videos as resources(This is a personal preference. You can delete or change it other type of resources such as textbooks) 
* For Each Week, assign a deliverable that focuses on implementing the topic covered in that week.

------------------------------------------------------------------------------------------------------------------------------

Always ask questions about the topic to the LLM. You can also ask it create quizzes to test your understanding.
